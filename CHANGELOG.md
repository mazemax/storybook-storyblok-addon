# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-20

### Added
- Initial public release
- Toolbar button for quick copy-to-clipboard functionality
- Panel UI with component metadata display
- Bookmarklet for transferring blocks to Storyblok
- Automatic field population from Storybook stories
- Support for component variants and custom configurations
- JSON preview and formatting
- Block instance generation with UUID support
- Field type mapping from Storybook controls to Storyblok fields
- Comprehensive documentation and examples

### Features
- One-click copy of component blocks from Storybook
- Visual editor integration with Storyblok
- Automatic prop extraction and field mapping
- Support for custom block names and field mappings
- Browser compatibility (Chrome, Firefox, Safari)
